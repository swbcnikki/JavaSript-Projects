Responsive Text

The above line under the heading is an example of a horizontal rule being used in html

Size of the window affects the size of the text.It responds to changes

Use the "vw" unit when sizing the text. 12vw will set the size to 12% of the viewport width.

Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50cm wide, 1vw is 0.5cm.
